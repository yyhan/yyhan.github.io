npm入门指南

NPM是随同NodeJS一起安装的包管理工具，能解决NodeJS代码部署上的很多问题，常见的使用场景有以下几种：

  • 允许用户从NPM服务器下载别人编写的第三方包到本地使用。
  • 允许用户从NPM服务器下载并安装别人编写的命令行程序到本地使用。
  • 允许用户将自己编写的包或命令行程序上传到NPM服务器供别人使用。

由于新版的nodejs已经集成了npm，所以之前npm也一并安装好了。同样可以通过输入 “npm -v” 来测试是否成功安装。命令如下，出现版本提示表示安装成功:

常用命令

npm init 初始化package.json文件

npm install <package_name> 安装依赖到当前目录，如果已经在package.json里配置好了一切依赖，只需要运行npm install，npm会自动安装package.json里指定的依赖

npm install <package_name> -g 安装依赖到全局环境，一般用于一些js工具或命令类的安装包

npm install <package_name> --save 安装依赖到当前目录，并将该依赖添加到package.json 文件的依赖(dependencies)项中

npm install <package_name> --save-dev 安装依赖到当前目录，并将该依赖添加到package.json 文件的开发环境依赖(devDependencies)项中

npm uninstall <package_name> 移除依赖
